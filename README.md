# HR Analytics – Employee Attrition Prediction & Workforce Intelligence Dashboard

## Project Overview

Employee attrition is one of the biggest challenges faced by organizations, impacting productivity, employee morale, and recruitment costs. This project leverages **Python, SQL, Machine Learning, and Power BI** to analyze employee data, identify the key drivers of attrition, predict employees at risk of leaving, and provide actionable business recommendations for improving workforce retention.

---

## Business Problem

Organizations often struggle to understand why employees leave and which factors contribute most to employee turnover.

This project aims to:

* Identify key employee attrition drivers.
* Analyze workforce trends using data analytics.
* Predict employee attrition using Machine Learning.
* Support HR teams with data-driven decision-making.
* Build interactive dashboards for workforce monitoring.

---

## Project Objectives

* Perform data cleaning and preprocessing.
* Conduct Exploratory Data Analysis (EDA).
* Build predictive Machine Learning models.
* Compare model performance using evaluation metrics.
* Analyze HR data using SQL.
* Create interactive Power BI dashboards.
* Generate business recommendations to improve employee retention.

# Key Business Insights

* Overtime significantly increases employee attrition.
* Lower job satisfaction is strongly associated with higher turnover.
* Employees with fewer years at the company have a greater likelihood of leaving.
* Salary and work-life balance are major factors influencing employee retention.
* Certain departments consistently experience higher attrition rates.
